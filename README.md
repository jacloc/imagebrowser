## Image Browser App

### Local Run Instructions
1. Add your Flickr API key to your Gradle local.properties file like so:  `FLICKR_API_KEY=your_api_key`.  (This is to make sure your API key doesn't get leaked by committing it in code)
2. Run app normally through Android Studio
